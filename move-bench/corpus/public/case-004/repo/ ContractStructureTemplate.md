# Smart Contract Structure Template

This template outlines the recommended structure and ordering for smart contract files.

---

## CONSTANTS

> Define all constant values used in the contract.

---

## ERRORS

> Declare custom errors that the contract may use.

---

## EVENTS

> List of events emitted by the contract.

---

## STATES

> Define state variables or storage data.

---

## FUNCTIONS

- CONSTRUCTOR FUNCTION
- ENTRY FUNCTIONS
- VIEW FUNCTIONS
- FRIEND FUNCTIONS
- INTERNAL FUNCTIONS
- TEST ONLY FUNCTIONS

---

This template defines a consistent and organized layout for smart contract files.  
Use the following comment block for each function section and place the relevant functions inside it:

--- 

```bash
    /* -----------------------------------------------------------------------------
    --------------------------------------------------------------------------------
    --------------------------------------------------------------------------------
                                    [NAME]
    --------------------------------------------------------------------------------
    --------------------------------------------------------------------------------
    ----------------------------------------------------------------------------- */
```
